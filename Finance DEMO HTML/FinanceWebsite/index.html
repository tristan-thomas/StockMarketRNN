Recursive Nueral Network Stock Market Analysis

Abstract

The first tasks we were involved in can be classified as “processing” data for our RNN. To prepare a matrix for our RNN we created 3 separate functions. Our first function which we named TrendLines will take in a dataframe and output k trend lines as a dictionary. Our input dataframe is extracted from the Yahoo Finance suite inside python. The resulting dataframe from TrendLines can be pushed forward as an input into our second function which we named MACD, this function outputs a Dataframe which is the input appended with a MACD column. The resulting DataFrame from MACD is then used as an input for a third function named SandR. This function outputs a DataFrame with another appended column containing support and resistance data and another column containing euclidean distances from the peaks and valleys. This Final DataFrame will be used in our RNN. We runn a RNN with multiple hidden LSTM layers using this dataframe to predict the future movement of the stock.

Methadology for feature creation

For our first function TrendLines, we ran a hill climbing algorithm (outputs a dictionary), and then predicted trendlines for all the peaks and valleys. For our Second Function MACD we simply calculated the MACD values at each timestamp. For our third function, we ran a hill climbing algorithm again,and then used the dictionary output to run through the peaks and valleys with a 5% bucket to find “repeated” values and define proper resistance and support points. Then we ran through the entire dataset and found the euclidean distance from the nearest peak and nearest valley. For all functions we appended the computed data to the input DataFrame and returned it.
